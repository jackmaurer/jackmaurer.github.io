Projects

Right now, I'm working on a cross-platform calendar application that's going to have lots of useful features you won't find in leading calendar apps. It's tentatively scheduled for release in April 2020. Until then, you can read the latest updates here.

Crossword Tiles

Nine-Card Golf in Python

In January 2020, I wrote a command-line version of a card game called nine-card golf. You can read all about it in this blog post or explore the source code here.

Virtual Genetics Lab
